# J.A.R.V.I.S — Just A Rather Very Intelligent System

A personal AI agent and holographic assistant built exclusively for Krishna Gupta, designed to mimic the voice interaction, persistent self-learning memory, and visual HUD features of the classic Iron Man system.

## Project Brief

J.A.R.V.I.S is engineered as a full-stack application using the MERN stack (MongoDB, Express, React, Node.js) to deliver secure and responsive cognitive capabilities:

- **AI Brain**: Powered by Google Gemini 2.0 Flash (with web search grounding) and backed by Groq Llama 3.3 for seamless fallback protection.
- **Holographic HUD**: Cinematic glassmorphic dashboard featuring SVG Arc Reactor state animation and real-time circular FFT audio spectrum graphs.
- **Interactive Audio**: Hands-free background monitoring using voice wake word detection ("Hey Jarvis") and hardware-level DSP double-clap triggers.
- **Cognitive Memory**: Persistent long-term memory node tracking, session telemetry, and automated user profiling.
- **Secure Access**: Device fingerprint authorization checks and encrypted security PIN verification.

## Architecture

This project consists of two core spaces:
- `/backend`: Node.js & Express server handling secure model routing, cryptography, and connection to MongoDB.
- `/frontend` *(in planning)*: React & Vite client providing the holographic dashboard.
